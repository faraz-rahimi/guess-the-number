# 🎯 Guess the Number

A simple **Guess the Number** game built with **HTML, CSS, and JavaScript**.

## 📖 About

The computer randomly generates a number, and the player tries to guess it. After each guess, the game provides feedback until the correct number is found.

## ✨ Features

* Random number generation
* User input validation
* Instant feedback
* Win/Lose messages
* Responsive design
* Clean and simple interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM & Math.random)

## 📚 What I Practiced

* DOM Manipulation
* Event Listeners
* Conditional Statements (`if / else`)
* `Math.random()`
* User Input Handling
* Basic JavaScript Logic
